1. Open the code folder in Visual Studio Code.

2. Open a Terminal, and run the command below. This installs all NPM packages referenced in the package.json file.

	npm install

3. Run the following installation commands for various other NPM packages:

	npm install @nestjs/config
	npm install @nestjs/graphql
	npm install @apollo/server @as-integrations/express5    (Note! together)

4. Run the project with the command below:

	npm run start:dev
